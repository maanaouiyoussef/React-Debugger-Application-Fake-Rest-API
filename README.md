# React-Debugger-Application-Fake-Rest-API
A simple React.js application that helps people to save the solution of their bugs.
## How To Use It ? 
### The application is using a fake rest api npm librairy that helps to create a fake backend
- Create a new empty project of react in your computer.
- npx create-react-app yourAppName
- git clone React-Debugger-Application-Fake-Rest-API
- Copy & paste the folders to your empty project. 
- npm i json-server react-icons
- npm run server
- npm start
## And Voila Enjoy it !
